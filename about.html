About This App

Rao's Chat X is a Firebase-powered, real-time chat application with a modern WhatsApp-style interface. Built for fast 1:1 conversations with advanced features like typing indicators, message reactions, online status, and more.

This application was created on August 20, 2025 as a demonstration of modern web development techniques and real-time communication capabilities.

Key Features

  • Real-time 1:1 messaging
  • Instant typing indicators
  • Message reactions
  • Online/offline status
  • User blocking system
  • Verified accounts
  • Profile avatars
  • Dark/Light theme

Technology Stack

Rao's Chat X is built with modern web technologies:

HTML5

Semantic markup

CSS3

Modern styling

JavaScript

Client-side logic

Firebase

Backend services

The application utilizes Firebase's complete suite:

  • Firebase Authentication for user management
  • Cloud Firestore for real-time data storage
  • Firebase Storage for media files
